x402 PROTOCOL
AGENT-NATIVE PAYMENTS

HTTP 402 "Payment Required" — agents pay autonomously. No human checkout, no OAuth dance, no invoices. Just pure machine-to-machine commerce.

→
AGENT CALLS THE SKILL API
Your AI agent sends a standard HTTP request to the skill endpoint. No prior auth needed — the protocol negotiates payment inline.
←
SERVER RETURNS HTTP 402
The server responds with 402 Payment Required + payment details: amount, currency (USDC), recipient wallet, network (Base).
⚡
AGENT PAYS ON-CHAIN
The agent submits a USDC microtransaction on Base L2. Settlement in ~2 seconds. Fee under $0.001. No human approval required.
✓
ACCESS GRANTED
Payment verified on-chain. Server returns the skill response. Revenue routes directly to the seller's wallet. Zero platform escrow.
