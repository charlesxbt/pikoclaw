# x402 on PikoClaw

## What is x402?

x402 is an open payment protocol by Coinbase that implements HTTP status code `402 Payment Required`. It enables instant micropayments for web services using USDC on Base L2.

## How it works

1. **Agent requests skill** — `GET https://api.pikoclaw.sh/skills/browse-agent`
2. **Server returns 402** with payment details:
   ```json
   {
     "scheme": "exact",
     "network": "base",
     "amount": "40000",
     "asset": "USDC",
     "payTo": "0xSellerWallet..."
   }
   ```
3. **Agent pays** — signs an EIP-3009 `transferWithAuthorization` for exact amount
4. **Agent retries** with `X-Payment: <signed-header>`
5. **Server returns 200** — skill delivered, receipt in `X-Payment-Receipt` header

## For Skill Sellers

Install the middleware:
```bash
npm install x402-express
```

Protect your routes:
```typescript
import { paymentMiddleware } from 'x402-express'

app.use(paymentMiddleware("0xYourWallet", {
  "GET /skills/browse-agent": { price: "$0.04", network: "base" }
}))
```

## For Buyers

Use any x402-compatible client:
- [x402-fetch](https://www.npmjs.com/package/x402-fetch) (JS/TS)
- [x402-python](https://pypi.org/project/x402) (Python)
- Any wallet that supports EIP-3009
